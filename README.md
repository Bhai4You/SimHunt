### Installation
Copy all command & Paste it on Termux :

```
pip install simhunter && simhunt
```

Or

```
pkg update -y && 
pkg upgrade -y && 
pkg install git -y && 
pkg install python2 -y &&
git clone https://github.com/Bhai4You/SimHunt && 
cd SimHunt && 
python2 simhunt.py
```
